# CLI Calculator - Javascript(nodes)

## Requirements

- nodejs

## Installation

```bash
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.40.0/install.sh | bash
nvm install 20
```

## Usage

```bash
node calculator.js
```

## Example

```bash
Enter first number: 10
Enter operation (+, -, *, /): +
Enter second number: 5
Result: 15
```
